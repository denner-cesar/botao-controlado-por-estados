 Botão controlado por estados

Este projeto demonstra como controlar o comportamento de um botão
utilizando uma máquina de estados finitos em JavaScript puro.

Por que esse projeto?
A maioria das interfaces controla ações com `if/else`.
Aqui, o fluxo é previsível, organizado e escalável usando estados e eventos.

 Estados possíveis
- PARADO
- CARREGANDO
- SUCESSO
- ERRO

 Tecnologias
- HTML
- CSS
- JavaScript (sem frameworks)
